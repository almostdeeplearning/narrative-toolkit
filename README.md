# BiteFlo

Turn X discussions into structured, easy-to-understand insights

![BiteFlo Preview](docs/preview.png)

---

I often save a lot of great posts on X, but find it really hard to absorb them right away.  
In the end, they just pile up in my bookmarks.

So I decided to build **BiteFlo** ——  
a semi-automated research tool designed for rich discussions on X.  

It helps turn professional posts and discussions on X into structured, easy-to-understand insights through AI workflows.

BiteFlo helps transform complex insights into understandable bites and structured knowledge, making it easier for non-experts to trace technology trajectories and evolving industry dynamics over time.

> Note: This tool is built mostly based on my own usage habits and is still evolving.

---

## 🧠 Core Philosophy

- **X-First**  
  Taking advantage of Grok’s ability to access X posts, BiteFlo streamlines the AI analysis process and quickly turns deep discussion threads into easy-to-understand insights.

- **AI-first, human-guided**  
  Let AI do the heavy lifting — extracting the latest opinions and organizing content — while users keep their mental energy for understanding, thinking, and interacting with AI.

- **Semi-automated Workflow**  
  Using multiple layers of prompts to connect  
  `capture → analyze → format → save`  
  into a streamlined flow, rather than just a prompt storage tool.

---

## ⚙️ Main Features

### 1. Narrative Scan  
**Quickly surface mainstream narratives on X**

- Use Grok to extract real-time discussions  
- Inject saved prompts to gather expert insights  
- Structure posts into usable formats  
- Manually refine and export as Markdown / HTML  

---

### 2. AI Flows  
**Deep analysis of individual posts**

- Build prompt workflows using step-by-step cards  
- Set delay timing for each step  
- Choose different AI models (Grok inline / Grok page / GPT)  
- Save workflows as presets and reuse them  

---

### 3. Prompt Manager

- Manage frequently used prompts with tab-based organization  
- Inline editing, series management, and expandable cards  
- Replace / merge import, JSON export, and Markdown export  

---

### 4. Format Manager

- Manage output formats (Markdown, YAML, Table, Wiki, etc.)  
- Inline editing  
- Replace / merge import, JSON export, and Markdown export  

---

### 5. Settings

- Theme switching (nt-dark / editorial-light / studio-light)  
- Font size and contrast adjustment  
- Download path and automation options  
- Chinese / English interface switching  

---

## 🚀 Installation

1. Go to `chrome://extensions/`  
2. Enable **Developer mode**  
3. Click **Load unpacked**  
4. Select the project folder
