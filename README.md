# BiteFlo

**Turn X discussions into structured, bitesized insights.**

I often save great posts on X but struggle to absorb them immediately.  
So I built **BiteFlo** — a semi-automated AI tool that helps transform complex X discussions into clear, structured knowledge.

> Note: This tool is built mostly based on my own usage habits and is still evolving. Expect some rough edges.

---

## 🧠 Core Features

- **Narrative Scan** — Quickly extract mainstream narratives and expert insights from X using Grok
- **AI Flows** — Build customizable multi-step prompt workflows (with delays, different models, and presets)
- **Prompt Manager** — Organized prompt library with series, inline editing, and import/export
- **Format Manager** — Multiple output formats (Wiki, YAML, Table, Markdown, etc.)
- **Settings** — Dark/Light themes, font size, language (中文/English), and more

---

## 🚀 Installation

1. Go to `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the project folder
